# Popup
## UI JSX 사용

`Popup` 컴포넌트는 `Portal`을 활용한 UI 입니다.

헤더, 바디, 푸터 모두 기본 스타일이 지정되어 있습니다.

<Story of={PopupStories.Default} />

```
<Popup isOpen={isOpen} onClose={() => setIsOpen(false)}>
  <PopupHeader>
    헤더 영역
  </PopupHeader>
  <PopupBody>
    팝업 콘텐츠 영역
  </PopupBody>
  <PopupFooter>
    푸터 영역
  </PopupFooter>
</Popup>
```


기본 적용 스타일

다른 스타일을 적용할 땐, style 을 덮어서 사용합니다.
```
export const modalHeader = css`
    display: flex;
    justify-content: space-between;
    font-size: 1.1em;
`;

export const modalBody = css`
    word-break: break-word;
    font-weight: 400;
`;

export const modalFooter = css`
    display: flex;
    justify-content: end;
    gap: 8px;
`;
```
