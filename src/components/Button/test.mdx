## variant
`variant` 속성으로 버튼의 모양을 바꿀 수 있습니다. 값으로는 `default`, `float`, `text`를 가진다. 따로 설정하지 않을 경우, default가 적용됩니다.


### EMCORE-TEST-0002

<Canvas of={ButtonStories.VariantDefault} />

### EMCORE-TEST-0003
<Canvas of={ButtonStories.VariantFloat} />

### EMCORE-TEST-0004
<Canvas of={ButtonStories.VariantText} />



## color
`color` 속성은 미리 선언된 버튼의 테마컬러를 선택할 수 있습니다. 현재 등록된 컬러 테마는 `primary`, `danger`, `default`

### EMCORE-TEST-0006

## Icon
`icon` 속성은 사용시 버튼에 아이콘과 함께 사용할 수 있습니다.

## Icon Position Right
`icon` 속성 사용시 기본 왼쪽에 위치한 아이콘을 오른쪽으로 위치합니다.

## Icon Only
`icon`을 사용하며 버튼 텍스트를 입력하지 않는다면 아이콘만 있는 작은 버튼을 사용할 수 있습니다.

## Loading
`loading` 속성 사용시 버튼 클릭으로 인한 동작이 활성화 된 상태를 시각화하며 버튼이 비활성화 됩니다.
