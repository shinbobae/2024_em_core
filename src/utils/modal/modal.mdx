# Util - modal

컴포넌트 외에도 모달을 렌더링 할 수 있도록 만들어졌습니다.

기본 모달과 확인창(Alert)을 열 수 있습니다.

컴포넌트와의 연계가 필요할 경우에는 컴포넌트 내에서 `useModal`, `useConfirm`, `<Popup />` 등을 사용하세요.

```
import modal from "@/utils/modal";

modal.openModal({ content: <div>Modal component2</div> });
modal.openAlert({
  title: '알립니다',
  content: '알림 메시지 입니다.',
  okText: '확인텍스트',
  ok: () => window.alert('확인')
});
```
